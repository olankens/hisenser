#### 2. Gather the Logos

```shell
deposit="images" && mkdir -p "$deposit"
baseurl="https://github.com/olankens/logobase/raw/HEAD/logos"
for f in {circle,horizontal,square,vertical}-{dark,light}; do
  curl -Lo "$deposit/$f.png" "$baseurl/$f.png"
done
```
